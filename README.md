# TRI Signature Generator — full mobile-safe

Генератор email-подписи TRI для GitHub Pages.

Эта версия сохраняет полную коммуникационную задачу:
- бренд TRI,
- описание компании,
- факты доверия,
- CTA «Подобрать решение»,
- опциональный блок «Дайджест TRI»,
- контакты.

При этом подпись сделана уже и ниже, чтобы лучше переживать Gmail mobile.

## Публикация

1. Создайте публичный репозиторий `tri-signature-generator`.
2. Загрузите в корень `index.html`, `README.md` и папку `assets`.
3. Включите GitHub Pages: Settings → Pages → Deploy from a branch → main → /root.
